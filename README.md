# 開発ネーム：ShootingYTAM
（よつあまって読んでほしいな〜）

## タイトル
* GooglePlay…フルチャージ!! -FullCharge!!-
* Androidアプリインストール後の表示名（日本語端末）…フルチャージ!!
* Androidアプリインストール後の表示名（日本語以外）…FullCharge!!

## ビルド設定
* CpmpanyName: YTAM（異論は認めるorz）
* ProductName: FullCharge!!
* CloudProjectid: jp

## 説明文例
### 簡単な説明

```
片手で遊べる縦スクロールシューティング！
ハイスコア目指して、
決めろ！フルチャージ！！
```

### 詳細な説明

```
【縦スクロールSTG】
   フルチャージ！ -FullCharge!!-

●敵の攻撃をかいくぐり、チャンスを狙ってショットを放て！
●チャージショットはためるごとに威力が倍増！
●オンラインランキング対応！トップを狙え！


Presented by nunu & yotsu

※このゲームは2015年度茨木高校文化祭数学研究部ゲーム展示にOB作品として出品予定です。
```
